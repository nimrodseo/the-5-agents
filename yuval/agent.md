# Yuval — Agent Pointer

Yuval's canonical agent definition lives at `.claude/agents/yuval.md`.

This folder (`yuval/`) is his working directory:

| Folder | Purpose |
|--------|---------|
| `reference/` | Drop inspiration images here. Yuval scans these before every generation to extract and apply visual style. |
| `outputs/` | Generated images land here, named `YYYY-MM-DD-<slug>.png`. Each PNG has a sibling `.txt` file with the exact prompt used. |

To invoke Yuval, ask the CEO Agent for an image (or use trigger keywords like "generate image", "תמונה של", etc.).
