# the-5-agents Vault

This vault is Claude Code's long-term memory for the **the-5-agents** content-creation team project. Per-file reference docs live in `Code Reference/`. Topical session logs live in the four content folders defined by [the obsidian-vault-workflow skill](../.claude/skills/obsidian-vault-workflow/SKILL.md).

> [!important] Workflow
> Every task in this project follows `obsidian-vault-workflow`. Read the relevant topic / reference file at the start, append a session log entry at the end. See [CLAUDE.md](../CLAUDE.md) at project root for the mandate.

## Folders

| Folder | Purpose |
|---|---|
| [[Code Reference/_index\|Code Reference]] | One file per project artifact — what it is, who owns it, related files |
| [[Meeting Notes/_index\|Meeting Notes]] | Code, architecture, decisions; session log per topic |
| [[Content Briefs/_index\|Content Briefs]] | Editorial briefs and campaign specs |
| [[Publishing Log/_index\|Publishing Log]] | Publish runs, outcomes, post-mortems |
| [[Brand Guidelines/_index\|Brand Guidelines]] | Voice, visuals, tone, UI primitives |
